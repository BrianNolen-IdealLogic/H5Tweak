# H5Tweak
Removes the 60 FPS cap, adds field of view customization and ultra-wide screen support for Halo 5: Forge.

# Requirements
* Halo 5: Forge
* .NET Framework 4.6.1
* [MemorySharp (lolp1 fork)](https://github.com/lolp1/MemorySharp/tree/Patterns)
